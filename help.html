Wireless Microphone Analyzer

The Wireless Microphone Analyzer shows the frequency spectrum captured via the RF Explorer or the TinySA (Ultra) hardware and overlays additional important information like forbidden frequency ranges, recommended/congested manufacturer channels and the old TV channel grid. It is thought to help audio engineers dealing with wireless microphones to easily discover free frequencies for their events.

The following overlays/selections for the live graph are available:

  • Wireless microphone vendors have defined specific frequency ranges for their equipment which have vendor specific names. The app allows to preselect these vendor specific bands. E.g. if you own a "Sennheiser WE 100 G3" in Sennheiser's E-Band, you can select exactly this frequency range to be shown on the screen.
  • Some vendors also offer pre-calculated intermodulation free channel maps for their equipment. If available these channel presets can be overlayed as well by selecting the corresponding vendor/device/channel map. Details like channel bank and channel number are shown on top of the chart in green.
  • Another overlay shows forbidden frequency ranges of a certain country (if available) in red. On top of the chart, additional information regarding a certain forbidden range is shown.
  • It is also possible to overlay the legacy TV channel grid. On top of the graph the TV channel numbers are shown. If the overlay is disabled, the TV channel numbers are still shown.

Connection

When starting the app, it automatically tries to detect the serial port (via USB) to which one of the supported scanning devices is attached to. If this doesn't work, open the "Port" menu and select the corresponding port manually. In case there is still no display, please restart the app and the device.

UI Description

The real-time graph works in "peak hold" mode. That means that the highest level detected for a certain frequency is kept. There is also a red line at -85dBm which commonly seems to be a maximum noise level which can be tolerated for wireless microphones. The peak line can be reset either via the menu or by pressing the "R" key.

It is possible to show/hide each of the following graph overlays:

  • Recommended Manuf. Channels
  • Forbidden Ranges
  • Congested / Forbidden Channels
  • TV Chan. Grid
